# ⚽ Latif Card Pro

Футбольное приложение с карточками игроков для FC Achinsk Academy.

## 🎮 Что умеет

- **📋 Коллекция** — собирай карточки игроков, фильтруй по позициям
- **📷 Фото** — загружай фото игроков с телефона или компьютера
- **⚡ Тренировка** — 6 мини-игр, каждая прокачивает свой навык
- **📈 Прокачка** — система XP и уровней, стата растёт от тренировок
- **✏️ Редактор** — создавай и редактируй любые карточки
- **🎨 Темы** — 6 тем оформления (Gold, Silver, Bronze, TOTW, Special, Hero)
- **💾 Авто-сохранение** — все данные сохраняются в браузере

## 🚀 Как запустить

### Шаг 1 — Установи Node.js

Скачай и установи с сайта: **https://nodejs.org** (кнопка LTS)

Проверь что установилось — открой терминал (командную строку) и напиши:
```
node --version
```
Должно показать версию, например `v20.11.0`

### Шаг 2 — Скачай проект

```
git clone https://github.com/1RamilRU/latif-app.git
cd latif-app
```

### Шаг 3 — Установи зависимости

```
npm install
```

### Шаг 4 — Запусти!

```
npm run dev
```

Откроется браузер на **http://localhost:3000** — приложение работает! 🎉

## 📁 Структура проекта

```
latif-app/
├── index.html              ← Главная HTML-страница
├── package.json            ← Зависимости проекта
├── vite.config.js          ← Настройки сборщика
├── src/
│   ├── main.jsx            ← Точка входа
│   ├── App.jsx             ← Главный компонент
│   ├── styles.css          ← Глобальные стили
│   ├── components/
│   │   ├── PlayerCard.jsx      ← Карточка игрока (лицо + оборот)
│   │   ├── CardViewScreen.jsx  ← Экран просмотра карточки
│   │   ├── CollectionScreen.jsx ← Экран коллекции
│   │   ├── TrainingScreen.jsx  ← Тренировки и мини-игры
│   │   ├── EditorScreen.jsx    ← Редактор карточек
│   │   ├── LevelUpModal.jsx    ← Анимация повышения уровня
│   │   ├── NavBar.jsx          ← Нижняя навигация
│   │   └── Particles.jsx       ← Фоновые частицы
│   ├── data/
│   │   ├── constants.js        ← Темы, позиции, константы
│   │   └── players.js          ← Дефолтные игроки
│   └── utils/
│       └── helpers.js          ← Вспомогательные функции
```

## 🛠 Технологии

- **React 18** — UI фреймворк
- **Vite** — быстрый сборщик
- **CSS-in-JS** — стили прямо в компонентах
- **localStorage** — сохранение данных
